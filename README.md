This is a work-in-progress Spotify Remote Player. Some features have not been implemented.
The project was made using React, Tailwind, nextAuth, and Recoil.

A video demo can be found here: https://youtu.be/VzeghVTIGGE


In order to view/use this web app please follow these directions:
1. Download necessary files and open them in editor.
2. Use 'npm install' to install necessary packages. 
3. Use 'npm run dev' to start development server.
4. Visit localhost:3000 in browser and sign in!
